📊 Customer Insights & Recommendation Dashboard
Overview

This project analyzes 100,000 customer transactions to uncover meaningful business insights and provide data-driven recommendations. The dashboard was built using Python for data cleaning and Power BI for interactive visualization.

Project Objectives
Analyze customer purchasing behavior
Identify top-performing product categories
Measure revenue, profit, and profit margin
Segment customers into High, Medium, and Low Value
Generate actionable business recommendations
Build an interactive Power BI dashboard
